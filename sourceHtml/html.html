HTML

HTML(HyperText Markup Language — «мова гіпертекстової розмітки») — базовий будівельний блок Інтернету. Він визначає зміст та структуру веб-контенту. Інші технології, крім HTML, зазвичай використовуються для опису зовнішнього вигляду/подання (CSS) або функціональності/поведінки (JavaScript) веб-сторінки.

Базові теги

!DOCTYPE html : Оголошення типу документа.
html: Елемент html містить весь вміст на всій сторінці, і іноді його називають "кореневий елемент".
head: Цей елемент виступає як контейнер для всього вмісту, який ви бажаєте включити в HTML документ, але не бажаєте показувати відвідувачам вашої сторінки.
meta charset="utf-8": Цей елемент встановлює як символьне кодування для вашого документа utf-8 , який включає більшість символів з усіх відомих людству мов.
title: Цей елемент встановлює заголовок вашої сторінки, який з'являється у вкладці браузера, що завантажує цю сторінку, також цей заголовок використовується при описі сторінки, коли ви зберігаєте її в закладках або вибраному. body: Він містить весь контент, який ви хочете показувати відвідувачам вашої сторінки, — текст, зображення, відео, ігри, аудіо доріжки, що програються, або щось ще.

Зображення

HTML-елемент img вбудовує зображення у документ. Це елемент, що заміщується.
Приклад тегу: <img src="./data/img.jpg" alt="Image">
Атрибут src є обов'язковим і містить шлях до зображення, яке ви хочете вбудувати в документ.
Атрибут alt містить текстовий опис зображення, який не обов'язково, але неймовірно корисний для доступності - програми читання з екрану читають цей опис своїм користувачам, так вони знають яке зображення показано, і так само воно відображається на сторінці, якщо зображення не може бути завантажене за якою або причини.
Aтрибут crossorigin вказує, чи слід використовувати CORS під час завантаження зображення чи ні.
Aтрибут decoding Надає рекомендації браузеру щодо декодування зображення. Допустимі значення:

  • sync: Декодувати зображення синхронно для одночасного відображення з іншим контентом;
  • async: Декодувати зображення асинхронно, щоб зменшити затримку відображення іншого контенту;
  • auto: Стандартний режим, який вказує на відсутність переваг до режиму декодування. Браузер вирішує, що найкраще для користувача.
Атрибут height внутренняя высота изображения в пикселях.
Атрибут width внутрішня ширина (див. Внутрішній розмір (en-US)) зображення в пікселях.
Атрибут importance Экспериментальная возможность. Указывает сравнительную важность ресурса. Приоритет выбирается с помощью значений:
  • auto: Указывает на отсутствие предпочтений. Браузер может использовать собственную эвристику для определения приоритета изображения;
  • high: Указывает браузеру, что изображение имеет высокий приоритет;
  • low: Указывает браузеру, что изображение имеет низкий приоритет.
Атрибут ismap це логічний тип, що вказує на те, що зображення є частиною серверної картки посилань. Якщо це так, то точні координати кліка надсилаються на сервер.
Примітка: Цей атрибут дозволений, лише якщо елемент img є нащадком елемента a з валідним (відповідним вимогам) атрибутом href.
